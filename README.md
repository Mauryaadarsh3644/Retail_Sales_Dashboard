# 🛒 Retail Sales Dashboard (Power BI Only)

## 📌 Project Overview

This project is a **Retail Sales Dashboard** built entirely using **Power BI**. It provides a comprehensive view of sales performance, profit trends, customer segmentation, and regional distribution.

The dashboard is designed for **business decision-making**, enabling users to quickly analyze key KPIs and identify growth opportunities.

---

## 🎯 Objectives

* Monitor overall sales and profit performance
* Identify top-performing products and brands
* Analyze sales distribution across regions and store types
* Track yearly and monthly sales trends
* Understand customer segments

---

## 🧰 Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Data Modeling (Relationships, Measures, Calculated Columns)**

---

## 📊 Dashboard Highlights

### 🔑 Key KPIs

* **Total Sales:** 4.72 Billion
* **Total Quantity:** 316K
* **Total Profit:** 874 Million
* **Profit Margin:** 18.49%

### 📈 Visual Insights

* **Sales Trend (Year-wise filter: 2019–2024)**
* **Sales vs Last Year Comparison (Monthly)**
* **Top 10 Products by Sales**
* **Top 5 Brands (Sales, Quantity, Profit)**

### 🌍 Distribution Analysis

* **Sales by Region:** North, South, East, West
* **Sales by Store Type:** Online, Offline, Franchise
* **State-wise Sales (Map Visualization)**

### 🎛 Interactive Features

* Year slicer (2019–2024)
* Customer segment filter
* Dynamic visuals with cross-filtering

## 📌 Key Insights

* Certain brands like **WorkWood and StackWell** generate high revenue
* **Online sales** contribute the highest share
* **South and North regions** dominate overall sales
* Seasonal spikes observed in mid-year months
